# Ergänzung: TTL-Auswahl integriert

- Neue/aktualisierte Datei: `formulare.html` (lädt TTLs und zeigt auswählbare IDs)
- Neuer Ordner: `data/` mit `dozentenblatt.ttl` und `zuarbeitsblatt.ttl`

## Start
Bitte Projekt über einen lokalen Webserver öffnen (z. B. Python 3):
```
python -m http.server 8000
```
Dann im Browser `http://localhost:8000/formulare.html` aufrufen.
